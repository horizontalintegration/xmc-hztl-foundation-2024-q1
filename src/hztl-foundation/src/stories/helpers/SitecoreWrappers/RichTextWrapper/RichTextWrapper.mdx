# Rich Text Wrapper

## Overview

RichTextWrapper presents content authored by content editors in a rich text format, similar to traditional "What you see is what you get" (wysiwyg) editors. It helps you render the content of the RichText Sitecore field type.

## Usage

Use it when you want to render wysiwyg generated content authored and managed by content editors.
