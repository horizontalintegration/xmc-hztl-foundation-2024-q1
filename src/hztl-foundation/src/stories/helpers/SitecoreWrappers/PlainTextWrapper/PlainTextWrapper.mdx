# `Plain Text Wrapper`

## Overview

PlainTextWrapper presents content authored by content editors in a simple text format. It helps you render the content of the Text Sitecore field type.

## Usage

Use PlainTextWrapper when you want to render Sitecore Single-Line Text, Multi-Line Text, or numeric field data.
