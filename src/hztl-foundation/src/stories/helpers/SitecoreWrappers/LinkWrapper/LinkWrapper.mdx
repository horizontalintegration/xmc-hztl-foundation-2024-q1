# `Link Wrapper`

## Overview

LinkWrapper presents an interactive element that triggers a route change when initiated by a user. It helps you render the content of the General Link Sitecore field type.

## Usage

Acts as a hyperlink

Supports internal and external URLs

You can use the Link component to render:

External links

Internal links, with HTML or other components

Email links

A content parameters link
